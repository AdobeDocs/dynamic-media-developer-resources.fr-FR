# Référence de l’API JavaScript pour Carousel Viewer{#javascript-api-reference-for-carousel-viewer}

La classe principale de la visionneuse de carrousel est CarouselViewer. Elle est déclarée dans l’espace de nommage s7viewers. Cette API JavaScript couvre le constructeur, les méthodes et les rappels de cette classe particulière.

Dans tous les exemples suivants, `<instance>` correspond au nom réel de l’objet de lecteur JavaScript appelé à partir de la classe `s7viewers.CarouselViewer`.
