# JavaScript référence de l’API pour la visionneuse Video360{#javascript-api-reference-for-video-viewer}

La classe principale du Video360Viewer est `Video360Viewer`. Elle est déclarée dans l’espace de noms s7viewers. Cette API JavaScript couvre le constructeur, les méthodes et les rappels de cette classe particulière.

Dans tous les exemples suivants, `<instance>` représente le nom réel de l’objet de visionneuse JavaScript instancié de la `s7viewers.Video360Viewer` classe.
