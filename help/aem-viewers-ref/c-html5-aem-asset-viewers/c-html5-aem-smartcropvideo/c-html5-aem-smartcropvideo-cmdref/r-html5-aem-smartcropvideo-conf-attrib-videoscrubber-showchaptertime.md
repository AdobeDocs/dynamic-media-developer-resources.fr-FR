# VideoScrubber.showchaptertime{#videoscrubber-showchaptertime}

Attribut de configuration de la visionneuse de vidéos avec recadrage intelligent.

`[VideoScrubber.|<containerId>_videoScrubber.]showchaptertime=0|1`

<table id="table_C616483932C2482CA9794DDD7313FD7C"> 
 <tbody> 
  <tr> 
   <td colname="col1"> <p> <span class="codeph"> 0|1</span> </p> </td> 
   <td colname="col2"> <p> Désactive ou active la durée du chapitre vidéo dans la barre de titre du libellé du chapitre vidéo. </p> </td> 
  </tr> 
 </tbody> 
</table>

## Propriétés {#section-f42369774e2740dcb399626a0e4e930e}

Facultatif.

## Par défaut {#section-d016470e92a74f98a18c4ab3489410a5}

`1`

## Exemple {#section-7621c8ebd4144bc08a537d01bd9c3f2f}

```
showchaptertime=0
```
