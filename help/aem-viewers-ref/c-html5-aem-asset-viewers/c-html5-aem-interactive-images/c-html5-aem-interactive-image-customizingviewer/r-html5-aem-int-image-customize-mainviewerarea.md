# Zone de visionneuse principale{#main-viewer-area}

La zone d’affichage principale est la zone occupée par l’image du zoom. Il est défini pour s’adapter à l’écran de l’appareil disponible lorsqu’aucune taille n’est spécifiée.

<!--<a id="section_061E550C1C1D4DB2BD663A898895B38C"></a>-->

**Propriétés CSS de la zone de visionneuse principale**

L’aspect de la zone d’affichage est contrôlé par le sélecteur de classe CSS suivant :

```
.s7interactiveimage
```

Exemple : pour configurer une visionneuse sur fond blanc ( `#FFFFFF`) et faire en sorte qu’elle mesure 1 174 x 500 pixels.

```
.s7interactiveimage { 
 background-color: #FFFFFF; 
 width: 1174px; 
 height: 500px;  
}
```
