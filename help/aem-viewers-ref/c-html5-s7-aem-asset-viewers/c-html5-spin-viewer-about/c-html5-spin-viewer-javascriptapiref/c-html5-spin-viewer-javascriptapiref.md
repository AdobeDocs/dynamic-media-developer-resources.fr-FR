# Référence de l’API JavaScript pour la visionneuse à 360°{#javascript-api-reference-for-spin-viewer}

La classe principale de la visionneuse à 360° est SpinViewer. Il est déclaré dans l’espace de noms s7viewers. Cette API JavaScript couvre le constructeur, les méthodes et les rappels de cette classe particulière.

Dans tous les exemples suivants, `<instance>` signifie le nom réel de l’objet de visionneuse JavaScript qui est instancié à partir de la classe `s7viewers.SpinViewer`.
