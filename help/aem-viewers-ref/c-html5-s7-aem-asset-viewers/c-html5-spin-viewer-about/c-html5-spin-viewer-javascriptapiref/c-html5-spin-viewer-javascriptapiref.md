# JavaScript référence de l’API pour Spin Viewer{#javascript-api-reference-for-spin-viewer}

La classe principale de la visionneuse à 360° est Visionneuse à 360°. Elle est déclarée dans l’espace de noms s7viewers. Cette API JavaScript couvre le constructeur, les méthodes et les rappels de cette classe particulière.

Dans tous les exemples suivants, `<instance>` représente le nom réel de l’objet de visionneuse JavaScript instancié de la `s7viewers.SpinViewer` classe.
