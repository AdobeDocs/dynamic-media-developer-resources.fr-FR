# Référence de l’API Javascript pour la visionneuse de zoom de base{#javascript-api-reference-for-basic-zoom-viewer}

La classe principale de la visionneuse de zoom de base est BasicZoomViewer. Elle est déclarée dans l’espace de nommage s7viewers. Cette API JavaScript couvre le constructeur, les méthodes et les rappels de cette classe particulière.

Dans tous les exemples suivants, `<instance>` correspond au nom réel de l’objet de lecteur JavaScript appelé à partir de la classe `s7viewers.BasicZoomViewer`.
