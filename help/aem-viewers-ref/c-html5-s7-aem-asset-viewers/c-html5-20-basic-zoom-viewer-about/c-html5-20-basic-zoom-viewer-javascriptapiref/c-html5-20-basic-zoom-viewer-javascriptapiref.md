# Référence de l’API JavaScript pour la visionneuse Zoom de base{#javascript-api-reference-for-basic-zoom-viewer}

La classe principale de la visionneuse Zoom de base est BasicZoomViewer. Il est déclaré dans l’espace de noms s7viewers. Cette API JavaScript couvre le constructeur, les méthodes et les rappels de cette classe particulière.

Dans tous les exemples suivants, `<instance>` signifie le nom réel de l’objet de visionneuse JavaScript qui est instancié à partir de la classe `s7viewers.BasicZoomViewer`.
