# ZoomView.reset{#zoomview-reset}

`[ZoomView.|<containerId>_zoomView.]reset=0|1`

<table id="table_49FFD1BC53B846F09A6D214BC8C5C3FE"> 
 <tbody> 
  <tr> 
   <td colname="col1"> <p> <span class="codeph"> 0 | 1</span> </p> </td> 
   <td colname="col2"> <p> Réinitialise le port de la vue lorsque le cadre (image) change. Si elle est définie sur <span class="varname"> 0</span>, elle préserve le port de vue actuel avec le meilleur ajustement possible tout en préservant les proportions de l'image nouvellement définie. </p> </td> 
  </tr> 
 </tbody> 
</table>

## Propriétés {#section-50bcd15223174bb79ce08b31ea03d682}

Facultatif.

## Par défaut {#section-7564169749ff4a4996049ea1148cb2a5}

`1`

## Exemple {#section-96e69b70365f461dae4399e49044ea2f}

`reset=0`
