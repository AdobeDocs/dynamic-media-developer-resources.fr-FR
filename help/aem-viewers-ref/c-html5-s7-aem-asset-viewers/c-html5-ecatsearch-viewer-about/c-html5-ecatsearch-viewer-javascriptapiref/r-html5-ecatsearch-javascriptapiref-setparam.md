# setParam{#setparam}

Référence de l’API JavaScript pour la visionneuse de catalogue électronique.

[!DNL ` setParam( *`nom, valeur`*)`]

Définit le paramètre de visionneuse sur une valeur spécifiée. Le paramètre est une option de configuration spécifique à la visionneuse ou un modificateur de kit de développement logiciel. Ce paramètre est appelé avant [!DNL `init()`].

Cette méthode est facultative si les informations de configuration de la visionneuse sont transmises avec [!DNL `config`] objet JSON au constructeur .

Voir aussi [init](../../../c-html5-s7-aem-asset-viewers/c-html5-20-ecatalog-viewer-about/c-html5-20-ecatalog-viewer-javascriptapiref/r-html5-ecatalog-viewer-20-javascriptapiref-init.md#reference-aee94dd92a28410784f7a1792e28683b).
