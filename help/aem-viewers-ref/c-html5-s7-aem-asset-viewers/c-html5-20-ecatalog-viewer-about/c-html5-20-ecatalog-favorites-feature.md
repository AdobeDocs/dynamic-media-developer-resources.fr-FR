# Fonctionnalité Favoris{#favorites-feature}

Un utilisateur final peut utiliser la fonction Favoris pour marquer les produits qui présentent un intérêt directement dans le catalogue électronique en ajoutant des icônes « cœur » aux images du catalogue électronique. La visionneuse stocke tous les favoris sur le système du client et les affiche à nouveau lorsque le même utilisateur consulte à nouveau le catalogue électronique.

En outre, la visionneuse vous permet de passer en revue tous les favoris ajoutés au catalogue électronique sous la forme d’une grille de miniatures.

Tous les outils Favoris sont regroupés dans une liste déroulante de la barre de contrôle.
