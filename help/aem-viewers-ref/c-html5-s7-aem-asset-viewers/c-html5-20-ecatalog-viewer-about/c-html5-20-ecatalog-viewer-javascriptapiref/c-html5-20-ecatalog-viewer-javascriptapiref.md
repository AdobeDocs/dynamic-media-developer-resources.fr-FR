# Référence de l’API JavaScript pour la visionneuse de catalogue électronique{#javascript-api-reference-for-ecatalog-viewer}

La classe principale de la visionneuse de catalogue électronique est eCatalogViewer. Elle est déclarée dans les  s7viewers . Cette API JavaScript couvre le constructeur, les méthodes et les rappels de cette classe particulière.

Dans tous les exemples suivants, `<instance>` représente le nom réel de l’objet de lecteur JavaScript appelé à partir de la `s7viewers.eCatalogViewer` classe.
