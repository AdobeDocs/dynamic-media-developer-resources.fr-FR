# Référence de l’API JavaScript pour la visionneuse de vidéos{#javascript-api-reference-for-video-viewer}

La classe principale de la visionneuse vidéo est VideoViewer. Elle est déclarée dans les  s7viewers . Cette API JavaScript couvre le constructeur, les méthodes et les rappels de cette classe particulière.

Dans tous les exemples suivants, `<instance>` correspond au nom réel de l’objet de lecteur JavaScript appelé à partir de la `s7viewers.VideoViewer` classe.
