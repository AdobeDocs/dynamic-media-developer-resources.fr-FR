# ZoomView.reset{#zoomview-reset}

`[ZoomView.|<containerId>_zoomView.]reset=0|1`

<table id="table_49FFD1BC53B846F09A6D214BC8C5C3FE"> 
 <tbody> 
  <tr> 
   <td colname="col1"> <p> <span class="codeph"> 0 | 1</span> </p> </td> 
   <td colname="col2"> <p> Réinitialise le port de  du lorsque le cadre (image) change. Si elle est définie sur <span class="codeph"> 0</span> , elle préserve le port  actuel avec le meilleur ajustement possible tout en préservant les proportions de l’image nouvellement définie. </p> </td> 
  </tr> 
 </tbody> 
</table>

## Propriétés {#section-65be9301796240e38f31818229da7acc}

Facultatif.

## Par défaut {#section-bd374ffc5182484faa77a7a3c8fa70f2}

`1`

## Exemple {#section-bd6c4249bccf44aab13fee8552f5a8b3}

`reset=0`
