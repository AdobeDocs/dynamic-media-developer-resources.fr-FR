# page{#page}

Récupérez la page. Récupère une page spécifique dans un fichier XFA de plusieurs pages.

`page= *`val`*`

<table id="simpletable_E92560F812B64A36A3D108CA7DEED5AC"> 
 <tr class="strow"> 
  <td class="stentry"> <p><span class="codeph"> <span class="varname"> val</span></span> </p> </td> 
  <td class="stentry"> <p>Numéro de page (la première page est 1). </p></td> 
 </tr> 
</table>

## Par défaut {#section-3fd7fcc525b947c7a95457e50414ac9e}

Si `page` n’est pas spécifié, la première page est renvoyée pour la sortie matricielle et toutes les pages pour la sortie PDF.
