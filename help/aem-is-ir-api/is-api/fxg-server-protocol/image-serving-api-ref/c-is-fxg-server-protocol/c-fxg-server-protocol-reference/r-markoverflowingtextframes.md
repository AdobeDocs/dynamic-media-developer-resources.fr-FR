# markOverflowTextFrames{#markoverflowingtextframes}

Afficher les blocs de texte en excès avec le signe plus. Un indicateur de débordement de texte s’affiche lorsque le texte dépasse l’espace qui lui est alloué dans un cadre de texte (ou dans le dernier cadre de texte dans le cas d’un texte lié). Cet indicateur est représenté sous la forme d’un cadre rouge avec un signe plus.

<table id="simpletable_F17FD29EB52043BF9000923ED5195A26"> 
 <tr class="strow"> 
  <td class="stentry"> <p><span class="codeph"> &amp;markOverflowTextFrames</span> </p> </td> 
  <td class="stentry"> <p>0|1 </p></td> 
 </tr> 
</table>

La définition de `markOverflowingTextFrames=1` de modificateur via un appel URL marque tous les blocs de texte dans lesquels le texte est surchargé par un signe plus. En outre, dans le prévisualiseur Dynamic Media Classic, l’indicateur de surréglage de texte est défini sur « `TRUE` » par défaut.

Valeur par défaut : 0.
