# imageRes{#imageres}

Définit la résolution des bitmaps incorporés dans une requête Web à imprimer.

` imageRes=< *`nombre`*>`

300 est recommandé lors de l’impression. Plus le nombre est élevé, plus le rendu de l’image prend du temps.
