# Nom{#name}

Nom de la macro.

Les noms de macro peuvent être constitués de toute combinaison de lettres ASCII, de chiffres, de caractères &#39;-&#39;, &#39;_&#39; et &#39;.&#39;.

## Propriétés {#section-3952a070c21646ab914cc76f289ed8ce}

Chaîne de texte. Obligatoire. Chaque valeur macro::Name doit être unique dans la table. Les noms des macros ne respectent pas la casse.

## Par défaut {#section-031d1931a7694da88c3f7d623eabcb78}

Aucune
