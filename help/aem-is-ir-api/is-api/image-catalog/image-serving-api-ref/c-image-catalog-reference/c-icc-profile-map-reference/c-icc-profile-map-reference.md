# Référence de la carte de profil ICC{#icc-profile-map-reference}

Si une carte de profil ICC est requise pour un catalogue d’images, le chemin relatif ou absolu du fichier de carte de profil doit être spécifié dans attribute::IccProfileMapFile.

Les entrées de la carte de profil ICC d’un catalogue d’images spécifique remplacent les entrées de la carte de profil ICC du catalogue par défaut.

Le service d’images prend en charge les fichiers de profil colorimétrique conformes à la spécification ICC.
