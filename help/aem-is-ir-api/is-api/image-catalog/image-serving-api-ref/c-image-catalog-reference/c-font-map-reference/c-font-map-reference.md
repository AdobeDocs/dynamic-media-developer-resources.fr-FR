# Référence de mappage de polices{#font-map-reference}

Si une correspondance de polices est requise pour un catalogue d’images, le chemin relatif ou absolu du fichier de correspondance de polices doit être spécifié dans attribute::FontMapFile.

Les entrées du mappage de polices d’un catalogue d’images spécifique remplacent les entrées du mappage de polices du catalogue par défaut.

Le service d’images prend en charge les polices Adobe Type 1, OpenType et TrueType.

`textPs=` prend également en charge les polices CID, ATC et bitmap.
