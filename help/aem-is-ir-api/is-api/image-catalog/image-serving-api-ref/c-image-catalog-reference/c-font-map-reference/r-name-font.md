# Nom{#name}

Nom de la police. Le nom de la police, exactement comme spécifié dans la commande RTF \fonttbl.

## Propriétés {#section-49d39cc6ec2b46c7a14e464dbe352f6a}

Chaîne de texte. Obligatoire. Clé d’index du Principal pour le mappage des polices. La même valeur font::Name peut apparaître jusqu’à quatre fois dans le même fichier de mappage de polices, mais chaque enregistrement doit avoir une combinaison différente de valeurs `font::Bold` et `font::Italic`.

## Par défaut {#section-2bcf8ad7a7934956ad2cd42fd0d401f8}

Aucune
