# Résolution{#resolution}

Résolution d&#39;objet par défaut. Fournit une résolution d&#39;objet par défaut au cas où un enregistrement de catalogue particulier ne contiendrait aucune valeur catalog::Resolution valide.

## Propriétés {#section-1f65ea81c3324781af17d0321facbd19}

Nombre réel, supérieur à 0. Généralement exprimé en pixels par pouce, mais peut également être exprimé dans d’autres unités, comme les pixels par mètre.

## Par défaut {#section-2d1258d2b7ce4929ac327a5d41a108dd}

Hérité de `default::Resolution` si non défini ou si vide.
