# PublishInfo{#publishinfo}

Chaîne d’annotation de catalogue. Chaîne de texte pouvant être utilisée pour identifier la version du catalogue, la date de publication, etc.

Le contenu de ce champ est renvoyé avec `req=catalogprops`. Cet attribut n’est pas utilisé autrement.

## Propriétés {#section-73ff2a7c2e5b4c7a952453276f5bed23}

Chaîne de texte.

## Par défaut {#section-82e5a885e142414f996c609efbb962b1}

Héritée de `default::PublishInfo` si non définie ou si vide.
