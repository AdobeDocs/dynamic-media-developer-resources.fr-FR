# RequestLock{#requestlock}

Mode de verrouillage de requête. Spécifie le type de verrouillage des demandes qui doit être inclus dans les requêtes.

## Propriétés {#section-ae89274d3b7940d0bf3d816f7c2f51fd}

Drapeau. Définissez sur 0 pour désactiver le verrouillage des demandes et sur 1 pour sélectionner un verrouillage de hachage simple.

## Par défaut {#section-fd5b5f0417bd444189b20494d179193e}

Hérité de `default::RequestLock` si non défini ou si vide.
