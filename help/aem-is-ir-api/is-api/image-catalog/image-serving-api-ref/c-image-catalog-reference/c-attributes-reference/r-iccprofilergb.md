# IccProfileRgb{#iccprofilergb}

 de couleurs de sortie RVB par défaut. Indique le nom du de couleurs ICC à utiliser pour les images de réponse RVB lorsqu’aucun espace colorimétrique de sortie n’est spécifié avec icc= et pour certaines valeurs de couleurs RVB spécifiées avec diverses commandes de diffusion d’images, telles que color=.

## Propriétés {#section-3dd55c954d4d4ad4bb715ed7cee31025}

Chaîne de texte. Le cas échéant, doit correspondre à une `icc::Name` valeur valide de la carte de  ICC de ce catalogue d’images ou du catalogue par défaut, ou à un chemin d’accès de fichier relatif à `attribute::RootPath`. Le ICC référencé doit être un RVB .

## Par défaut {#section-dfe08dd7b851453ca816623a4179955b}

Héritée de `default::IccProfileRgb` si non définie ou si vide.
