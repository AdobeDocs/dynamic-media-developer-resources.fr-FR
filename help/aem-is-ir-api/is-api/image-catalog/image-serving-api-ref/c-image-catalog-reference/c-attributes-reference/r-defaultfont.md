# DefaultFont{#defaultfont}

Nom de police par défaut. Indique la police à utiliser si aucune police n’est fournie par une requête text=.

## Propriétés {#section-b71618fec3c443a392d67eff480f0e5d}

Chaîne de texte. Si spécifié, doit être une valeur font::Name valide dans la liste de polices de ce catalogue d’images ou dans la liste de polices du catalogue par défaut.

## Par défaut {#section-ed0199be32a24e31a47b345ce7d109f6}

Hérité de `default::DefaultFont` si non défini ou si vide.
