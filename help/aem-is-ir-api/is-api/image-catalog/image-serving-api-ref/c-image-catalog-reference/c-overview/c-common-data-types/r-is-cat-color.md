# Couleur{#color}

Type de données de couleur.

Valeur de la couleur. Valeur RVB hexadécimale bondée, éventuellement précédée de `0x`. Par exemple, la valeur RVB `128,255,0` peut être spécifiée sous la forme `0x80ff00` ou `80ff00` .
