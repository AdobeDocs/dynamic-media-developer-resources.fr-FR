# Couleur{#color}

Type de données de couleur.

Valeur de couleur. Valeur RVB compressée hexadécimale, éventuellement précédée de `0x`. Par exemple, la valeur `128,255,0` RVB peut être définie sous la forme `0x80ff00` ou `80ff00` .
