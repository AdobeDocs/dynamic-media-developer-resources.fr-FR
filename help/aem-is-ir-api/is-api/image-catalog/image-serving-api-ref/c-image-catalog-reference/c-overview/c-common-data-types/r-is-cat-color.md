# Couleur{#color}

Type de données de couleur.

Valeur de la couleur. Valeur RGB hexadécimale compressée, éventuellement précédée de `0x`. Par exemple, la valeur RGB `128,255,0` peut être spécifiée comme `0x80ff00` ou `80ff00` .
