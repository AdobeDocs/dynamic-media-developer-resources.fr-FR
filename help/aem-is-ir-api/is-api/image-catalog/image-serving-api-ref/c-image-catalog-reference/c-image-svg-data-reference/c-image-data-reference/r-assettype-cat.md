# AssetType{#assettype}

Type de ressource. Utilisé pour identifier le type de visionneuse publiée dans catalog::ImageSet.

Le type de ressource détermine le type de réponse à générer pour les requêtes `req=set`. Si aucune valeur n’est spécifiée, les règles de détection automatique déterminent le type de réponse `req=set`.

## Propriétés {#properties}

Enum value de l’ensemble suivant :

MEDIASET

SPINSET

SPINSET2D

VIDEOSET

ENSEMBLE D’IMAGES

ECATALOGUE

## Par défaut {#section-b94ce61423194729918456103ef3f72c}

Aucune
