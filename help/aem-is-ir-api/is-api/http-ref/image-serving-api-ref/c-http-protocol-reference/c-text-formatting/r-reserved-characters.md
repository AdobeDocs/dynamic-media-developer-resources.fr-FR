# Caractères réservés{#reserved-characters}

Cette section répertorie les caractères réservés à des fins spécifiques.

RTF utilise des accolades &#39; `{`&#39; et &#39; `}`&#39; comme délimiteurs de groupe. Elles doivent se produire par paires et peuvent être imbriquées. Pour afficher les accolades dans les chaînes de texte, utilisez respectivement « `\{` » et « `\}` ».

>[!NOTE]
>
>Vous devez encoder en URL toutes les accolades utilisées en RTF.

Les barres obliques inverses &#39;\&#39; sont utilisées pour introduire les commandes et mots-clés RTF. Pour afficher une barre oblique inverse, utilisez `'\\'`.
