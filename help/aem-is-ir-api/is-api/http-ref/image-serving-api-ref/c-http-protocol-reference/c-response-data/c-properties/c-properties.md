# Propriétés{#properties}

Les données de propriété sont renvoyées en réponse aux types de `req=` suivants :

* `catalogprops`
* `imageprops`
* `props`
* `userdata`

`userdata` réponses ne sont formatées en tant que propriétés que lorsque le contenu des `catalog::UserData` suit le format des propriétés.

* [Propriétés de texte (Java)](r-text-java-properties.md)
* [Propriétés de JavaScript](r-javascript-properties.md)
* [Propriétés XML](r-xml-properties.md)
* [Propriétés JSONP](r-json-properties.md)
