# Calques nommés{#named-layers}

La commande layer= permet d’attribuer un nom à un calque et de référencer ultérieurement le même calque par son nom plutôt que par son numéro.

Cela permet à un modèle de définir des couches nommées qui peuvent ensuite être référencées à partir de l’URL de requête sans avoir à utiliser de variables.
