Les modes de matelassage aléatoire (14...18) peuvent être utilisés pour synthétiser des textures à partir d&#39;images qui ne sont pas facilement répétables ; l&#39;algorithme crée des textures entièrement aléatoires ou partiellement aléatoires sur la base de l&#39;image d&#39;origine.

## Propriétés {#section-262bf540930d4890b678ea00cefe1909}

Attribut Material. Ignoré par la couleur unie, la vignette et les matériaux de l&#39;armoire.

## Par défaut {#section-e5bbd7d9fbb74852849e605d20f550bb}

`catalog::Repeat`, si le matériau est basé sur une entrée de catalogue, sinon `0` (répétition directe).
