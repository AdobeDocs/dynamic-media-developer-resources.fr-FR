# masquer{#hide}

Masquez l’objet de chevauchement. Masque tous les objets de chevauchement ou les couvre-fenêtres appliqués aux cadres de couvre-fenêtres de la sélection en cours.

`hide[= *`val`*]`

 <tr class="strow"> 
  <td class="stentry"> <p><span class="codeph"> <span class="varname"> val</span></span> </p> </td> 
  <td class="stentry"> <p>Ignoré. </p></td> 
 </tr> 
</table>

## Propriétés {#section-2952fa43df0547b187b834c819c517ca}

S’applique au groupe ou à la sélection d’objets en cours. Ignoré si la sélection en cours ne contient pas d&#39;objets de chevauchement ou de cadres de revêtement de fenêtre ou si aucun objet de chevauchement n&#39;est visible.

## Par défaut {#section-9da202944d3a49b7ae81c78370b2b5e7}

S’il n’est pas spécifié, la visibilité actuelle des objets de chevauchement sélectionnés est conservée.
