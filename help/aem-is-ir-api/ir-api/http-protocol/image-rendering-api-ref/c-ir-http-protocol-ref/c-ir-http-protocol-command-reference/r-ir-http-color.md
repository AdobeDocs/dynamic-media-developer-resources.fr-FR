# couleur {#color}

Couleur de premier plan. Spécifie la couleur des matériaux de couleur unie ou la couleur additive pour les matériaux colorisables.

`color= *`color`*`

<table id="simpletable_C5AF9074CCA64EA5921772DF3F7E0F55"> 
 <tr class="strow"> 
  <td class="stentry"> <p><span class="varname"> Couleur</span> </p> </td> 
  <td class="stentry"> <p>Valeur de couleur RVB ou grise. </p></td> 
 </tr> 
</table>

## Propriétés {#section-629c3c91221c48c4b7f7b31a13fd1766}

Attribut matériel. Obligatoire pour les matériaux de couleur unie, facultatif pour tous les autres matériaux.

## Par défaut {#section-ea8e1967674d426bb8f46abe365b6aca}

`catalog::Color` Si le matériau est basé sur une entrée de catalogue. Sinon, `none` sans colorisation.
