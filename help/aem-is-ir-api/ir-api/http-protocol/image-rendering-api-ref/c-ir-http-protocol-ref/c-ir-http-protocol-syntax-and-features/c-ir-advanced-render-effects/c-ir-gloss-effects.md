# Effets de brillance{#gloss-effects}

Si cela est activé dans la vignette, le rendu fait varier l&#39;algorithme d&#39;illumination pour simuler une variation de la brillance du matériau (principalement en contrôlant le contraste).

Cet effet est indépendant du rendu de réflexion et le complète en simulant des réflexions spéculaires. L&#39;effet est contrôlé avec les mêmes attributs `gloss=`, `glossmap=` et `type=` que ceux utilisés pour contrôler les réflexions.
