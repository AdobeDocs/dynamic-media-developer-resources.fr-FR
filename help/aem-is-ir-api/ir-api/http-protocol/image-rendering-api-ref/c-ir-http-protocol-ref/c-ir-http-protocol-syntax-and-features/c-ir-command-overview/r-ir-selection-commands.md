# Commandes de sélection{#selection-commands}

Ces commandes permettent de sélectionner des groupes de vignettes, des objets, des sous-zones de groupes ou d’objets.

La commande, ou la matière, ou les deux, sont spécifiées après cette commande de sélection et avant l’application de la commande de sélection suivante (ou de la fin de la requête) à l’élément sélectionné. Les commandes de sélection délimitent les segments de spécification de matériau (MSS).
