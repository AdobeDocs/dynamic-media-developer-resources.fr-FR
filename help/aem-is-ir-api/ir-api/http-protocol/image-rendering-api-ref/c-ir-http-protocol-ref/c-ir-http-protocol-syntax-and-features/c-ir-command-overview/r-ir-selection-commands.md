# Commandes de sélection{#selection-commands}

Ces commandes permettent de sélectionner des groupes de vignettes, des objets, des sous-zones de groupes ou d’objets.

La commande, ou le matériau, ou les deux, sont spécifiés après cette commande de sélection et avant que la commande de sélection suivante (ou la fin de la requête) ne soit appliquée à l’élément sélectionné. Les commandes de sélection délimitent les segments de spécification de matériau (MSS).
