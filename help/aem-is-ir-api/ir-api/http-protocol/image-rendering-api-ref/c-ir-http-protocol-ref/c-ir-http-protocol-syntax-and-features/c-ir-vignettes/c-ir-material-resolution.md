# Résolution matérielle{#material-resolution}

La valeur de résolution d&#39;une texture répétable ou d&#39;un autre matériau définit la taille de l&#39;image du matériau dans l&#39;espace de coordonnées de la scène.

Les valeurs de résolution de texture sont spécifiées en pixels par pouce (ou selon les unités dictées par la vignette pour son espace de coordonnées de scène).

La résolution des matières est spécifiée par `res=`.
