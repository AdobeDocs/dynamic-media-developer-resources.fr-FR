# Options des fichiers de style armoire{#options-for-cabinet-style-files}

Les options suivantes contrôlent le traitement des fichiers de type armoire. Ils sont ignorés si le fichier sourceFile n&#39;est pas un fichier de style CAB ou si -info est spécifié.

<table id="simpletable_332B78DDEB6540708844AB54AE321F9B"> 
 <tr class="strow"> 
  <td class="stentry"> <p><span class="codeph">-résolution <span class="varname"> val</span></span> </p> </td> 
  <td class="stentry"> <p>Spécifie la résolution cible pour les fichiers de style armoire. Valeur réelle en pixels par pouce. S'il n'est pas spécifié, le fichier de style d'armoire de production a la même résolution que le fichier source. </p></td> 
 </tr> 
</table>
