# Renvoie{#returns}

vntc renvoie les valeurs suivantes, selon qu’une erreur est survenue pendant le traitement ou non.

* 0 si aucune erreur n’est survenue.
* 1 si des erreurs (fatales ou non) sont rencontrées.
