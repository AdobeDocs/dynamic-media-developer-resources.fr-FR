# MacroFile{#macrofile}

Chemin d&#39;accès au fichier de définitions de macro. Spécifie le fichier contenant les définitions de macro pour ce catalogue de matières.

## Propriétés {#section-74ab431f12fb4fc9b1e222c752cfb4cb}

Valeur de chaîne de texte. Chemin d’accès absolu au fichier ou chemin d’accès relatif au dossier du catalogue.

## Par défaut {#section-1ceb724a30444a5fbb870105b83c212c}

Vide, ce qui indique que ce catalogue de matières ne comporte pas de définitions de macro privées.
