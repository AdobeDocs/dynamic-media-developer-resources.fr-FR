# AllowDirectURLs{#allowdirecturls}

Autoriser les URL sources absolues.

Définissez cette variable sur 1 pour autoriser les URL HTTP absolues (commençant par &quot; `http://`&quot;) dans `src=` les commandes. Définissez cette variable sur 0 pour interdire les URL absolues.

## Propriétés {#section-9246bc06c7484f9e8f65a5edb24f6854}

Drapeau.

## Par défaut {#section-a881c30ea28e47e296faa890c3bd6daf}

Héritée de `default::AllowDirectUrls` si non définie ou si vide.
