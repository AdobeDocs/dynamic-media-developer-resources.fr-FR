# IccProfileMapFile{#iccprofilemapfile}

Chemin d&#39;accès au fichier de carte de profil ICC. Indique le fichier contenant la carte de profil ICC pour ce catalogue d’images.

## Propriétés {#section-33d55b55e192441bb30ca5eaaf07d12a}

Valeur de chaîne de texte. Chemin d’accès absolu au fichier ou chemin d’accès relatif au dossier du catalogue.

## Par défaut {#section-191de4cf304544f0a4198ebd3b8ddfc2}

Vide, ce qui indique que ce catalogue de matériaux n&#39;inclut pas les profils ICC privés.
