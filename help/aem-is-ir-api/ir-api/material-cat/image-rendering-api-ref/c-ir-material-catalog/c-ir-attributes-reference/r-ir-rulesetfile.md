# RuleSetFile{#rulesetfile}

Chemin d’accès au fichier des définitions de règles. Spécifie le fichier qui contient les définitions d&#39;ensemble de règles pour ce catalogue de matières.

## Propriétés {#section-e805898763824caba24abb6af1706f5c}

Valeur de chaîne de texte. Chemin d’accès absolu au fichier ou chemin d’accès relatif au dossier du catalogue.

## Par défaut {#section-69b1f865aa8d476bb087dc862e903ab1}

Vide, ce qui indique que ce catalogue de matières ne comporte pas de règles de prétraitement des demandes privées.
