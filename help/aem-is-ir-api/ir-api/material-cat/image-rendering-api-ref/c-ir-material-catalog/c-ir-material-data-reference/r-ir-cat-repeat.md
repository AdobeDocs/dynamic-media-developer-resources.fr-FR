# Répéter{#repeat}

Mode de répétition de la texture. Indique comment les images de texture sont mosaïquées pour remplir la surface cible.

## Propriétés {#section-cef4109cddf54ce095c3293d85bc412d}

Énumération. Utilisé uniquement pour les textures répétables. Ignoré pour tous les autres matériaux.
