# Référence des données matérielles{#material-data-reference}

Si des données de matières sont requises pour un catalogue de matières, le chemin relatif ou absolu du fichier de données de matières doit être spécifié dans attribute::CatalogFile.

Les entrées du fichier de données de matières d&#39;un catalogue de matières spécifique remplacent les entrées du fichier de données de matières du catalogue par défaut.
