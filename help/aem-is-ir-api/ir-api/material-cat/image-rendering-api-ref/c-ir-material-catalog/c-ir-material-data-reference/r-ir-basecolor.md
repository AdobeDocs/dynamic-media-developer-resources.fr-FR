# BaseColor{#basecolor}

Couleur d’image dominante. Couleur soustractive, utilisée pour la coloration des textures. Doit être la couleur dominante moyenne dans l’image de texture.

Vous pouvez également utiliser `BaseColor`comme couleur de remplissage si un matériau de texture sans `catalog::Color` est appliqué à un objet vignette non texturable.

## Propriétés {#section-d9aec235f28b402581cf2277f5fc6fbb}

Couleur. Ignoré par la couleur unie et les matériaux de l&#39;armoire.

## Par défaut {#section-ca1813a6fcb24645b5835af0891b279d}

`0x808080` (gris neutre).
