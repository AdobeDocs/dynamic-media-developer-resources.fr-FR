# Démarrage ou arrêt sous Windows{#starting-or-stopping-on-windows}

Il existe plusieurs mécanismes pour démarrer ou arrêter le serveur d’images et le [!DNL Platform Server] sous Windows.

1. Sélectionnez l’une des options suivantes pour démarrer ou arrêter la diffusion d’images :

* Le Panneau de Contrôle de service Windows peut être utilisé pour démarrer, arrêter, redémarrer et vérifier le statut du service `Dynamic Media Image Serving`.
* **[!UICONTROL Menu Démarrer]**> **[!UICONTROL Tous les programmes]**> **[!UICONTROL Dynamic Media (Scene7)]**> **[!UICONTROL Diffusion d’images]**> **[!UICONTROL Service de contrôle]** fournit des commandes pour démarrer, arrêter et redémarrer le service `Dynamic Media Image Serving`.
