# Démarrage ou arrêt sous Windows{#starting-or-stopping-on-windows}

Il existe plusieurs mécanismes pour démarrer/arrêter Image Server et Platform Server sous Windows.

1. Sélectionnez l’une des options suivantes pour démarrer ou arrêter la diffusion d’images :

* Le Panneau de Contrôle de service Windows peut être utilisé pour démarrer, arrêter, redémarrer et vérifier l’état du service &quot;Dynamic Media Image Serving&quot;.
* **[!UICONTROL Menu Démarrer]** >  **[!UICONTROL Tous les programmes]** >  **[!UICONTROL Dynamic Media (Scene7)]** >  **[!UICONTROL Image Serving]** >  **[!UICONTROL Control]** Service fournit des commandes pour Démarrer, Arrêter et Redémarrer le service Dynamic Media Image Serving.
