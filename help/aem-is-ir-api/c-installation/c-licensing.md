# Octroi de licence{#licensing}

Pour utiliser la diffusion d’images, une clé de licence de diffusion d’images Dynamic Media est requise et doit être installée sur chaque système sur lequel la diffusion d’images doit être utilisée.

De même, pour utiliser le rendu d’image, une clé de licence de rendu d’image Dynamic Media est requise. Le rendu d’image peut être utilisé sans licence de diffusion d’images, et inversement.

La procédure d’installation comprend les étapes permettant d’obtenir et d’installer les clés de licence nécessaires.
