# copyImage{#copyimage}

Crée une copie d’une ressource d’image existante. Les commandes de protocole du serveur d’images spécifiées sont appliquées pour générer la nouvelle copie

Syntaxe

## Types d’utilisateurs autorisés {#section-c9fe7abb550e495f832234f845db7d6e}

* `IpsUser`
* `IpsAdmin`
* `IpsCompanyAdmin`
* `ImagePortalAdmin`
* `ImagePortalContrib`
* `ImagePortalContribUser`

## Exemples {#section-c30a4017001146e7befbbfc5ffcb7593}

L’exemple de code copie une image spécifiée par la société, la ressource, l’identificateur de dossier et le nom.

**Requête**

```java
<copyImageParam xmlns="http://www.scene7.com/IpsApi/xsd/2008-01-15">
   <companyHandle>c|6</companyHandle>
   <assetHandle>a|739|1|537</assetHandle>
   <folderHandle>ApiTestCo/</folderHandle>
   <name>Copy_macbookwin1</name>
   <urlModifier/>
</copyImageParam>
```

**Réponse**

```java
<copyImageReturn xmlns="http://www.scene7.com/IpsApi/xsd/2008-01-15">
   <assetHandle>a|943|1|580</assetHandle>
</copyImageReturn>
```
