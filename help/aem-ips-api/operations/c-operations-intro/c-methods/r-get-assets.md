# getAssets{#getassets}

Renvoie les actifs d’IPS.

Syntaxe

## Types d’utilisateur autorisés {#section-4673c1c9f4314160af8b165eb2dd20cc}

* `IpsUser`
* `IpsAdmin`
* `IpsCompanyAdmin`
* `TrialSiteAdmin`
* `TrialSiteUser`
* `ImagePortalAdmin`
* `ImagePortalUser`
* `ImagePortalContrib`
* `ImagePortalContribUser`

>[!NOTE]
>
>Renvoie uniquement les actifs auxquels l’utilisateur a accès.
