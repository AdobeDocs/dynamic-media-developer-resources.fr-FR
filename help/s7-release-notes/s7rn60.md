# Notes De Mise À Jour De Scene7 Publishing System 6.0{#scene-publishing-system-release-notes}

Notes de mise à jour qui décrivent les fonctionnalités, les améliorations et les correctifs de la version 6.0 d’Adobe Scene7 Publishing System dans Adobe Marketing Cloud.

   * Catalogues électroniques

   * Série d’échantillons
   * Spin
   * Découpe vidéo

* Schémas de métadonnées

  Les schémas de métadonnées définissent des ensembles de propriétés de métadonnées. Chaque propriété du schéma de métadonnées possède un type de données de base tel que Chaîne, Flottant, Entier, Date ou Oui/Non. Vous pouvez également définir des listes, des choix ouverts ou des choix fermés. Les schémas de métadonnées peuvent être utilisés dans tous les workflows de métadonnées tels que la recherche, la modification, l’importation, les vues de métadonnées et les paramètres prédéfinis de métadonnées. <!-- [More information](http://help.adobe.com/en_US/scene7/using/WS259993e42159a215-1c6a66df1265272619e-7fec.html#WSd968ca97bf03cf72-5e3dd3a113268dc80f5-8000). -->

  Les schémas de métadonnées sont la technologie qui succède aux FDU et sont destinés aux workflows de métadonnées personnalisés, en particulier dans le contexte du serveur de métadonnées.

* Serveur de métadonnées

  >[!NOTE]
  >
  >Le serveur de métadonnées est disponible à partir de mai. Il doit être activé par le support. Le serveur de métadonnées est actuellement disponible uniquement en Amérique du Nord et en Europe.

* Le serveur de métadonnées expose un sous-ensemble de tous les champs de métadonnées d’une ressource à un serveur public et peut être demandé ici par des pages ou des serveurs eCom. Ces métadonnées exposées peuvent être un ensemble de propriétés de ressource, de champs définis par l’utilisateur, de XMP et de propriétés de schéma de métadonnées. Le serveur de métadonnées offre une API WEB avec diverses fonctionnalités de recherche telles que la recherche de métadonnées, la similarité d’images ou les deux. Pour chaque ressource renvoyée, une liste de propriétés de métadonnées est demandée.
* L’importation de métadonnées peut gérer les propriétés XMP

  L’importation de métadonnées est étendue pour gérer les propriétés XMP, définies dans les schémas de métadonnées.
* Vue de métadonnées pour afficher uniquement les champs contenant des valeurs

  Il existe une nouvelle vue de métadonnées prédéfinie appelée « Toutes les propriétés avec des valeurs » qui affiche uniquement les propriétés de métadonnées contenant des valeurs. Les propriétés (de XMP, des champs définis par l’utilisateur ou des nouveaux schémas de métadonnées) sans valeurs sont masquées et ne s’affichent pas.
* Paramètres de métadonnées désormais regroupés dans l’interface utilisateur d’administration.

  Dans **[!UICONTROL Configuration > Configuration de l’application]**, les pages de paramètres liées aux métadonnées sont désormais regroupées dans une section de niveau supérieur appelée [!DNL Metadata]. Vous y trouverez les pages de paramètres de la vue de métadonnées, des paramètres prédéfinis de métadonnées, du schéma de métadonnées et des champs définis par l’utilisateur.

Cette application de bureau vous permet d’importer des modèles 3D existants et de les créer sous forme de fichiers de vignette 2D. Les fichiers de vignette résultants peuvent être téléchargés sur SPS, où les couleurs et les textures peuvent être remplacées à la demande.

## Correctifs logiciels {#section-4219d5dd0a024cd3a95aaac45598363c}

Les problèmes suivants ont été corrigés dans le serveur de métadonnées :

* Correction d’un problème en raison duquel les chargements de fichiers EPS très volumineux échouaient.
* Les « recherches de métadonnées » avec une longue liste de résultats échouent silencieusement. Ce problème entraîne l’affichage de résultats de recherche incorrects.
